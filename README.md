# mslab
微服务实验室

### msrc
Micro Service Registary Center - 微服务注册中心
端口：8761

### mscc
Micro Service Config Center - 微服务配置中心
端口：8888

### stock
提供股票行情查询服务，服务名：mslab-stock

### antxchecker
提供antx配置文件的扫描服务，服务名：mslab-antxchecker

### filecrawler
提供目录文件查找统计服务，服务名：mslab-filecrawler

### mslab-support
基础构建封装
#### core-common
核心公共模块，包含了公共的工具类、枚举类
#### swagger-support
swagger工具控件
